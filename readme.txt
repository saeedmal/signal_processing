The main source code is in the package group_magnetic_sensor_data.py

The python version tested for this code is 3.12.2
The package versions used for analysis are listed in the requirements.txt

You just need to make a new python virtual environment

- python3 -m pip install --upgrade pip
- python3 -m venv venv-ommo
- python3 -m pip install -r requirements.txt

and then go through the steps in the jupyter notebook of analyze_magnetic_data.ipynb
